# System Overview / 系统概述

## 中文 (Chinese):

### 项目目标
MRI AutoQA 旨在提供一个自动化、标准化的 MRI 影像质量分析解决方案，帮助医疗机构和研究人员评估 MRI 影像的质量指标。

### 主要功能
- **自动化质量分析**: 一键运行批量 MRI 影像质量评估
- **多维度质量指标**: SNR、CNR、噪声均匀性、伪影检测等
- **解剖区域适配**: 支持脊柱、脑部、关节等多种解剖区域
- **序列类型识别**: 自动识别 T1、T2、PD、FLAIR 等序列类型
- **可视化报告**: 生成详细的质量分析可视化报告
- **标准化输出**: 统一的数据格式和报告结构

### 适用场景
- 医疗机构影像质量控制
- 临床试验影像质量评估
- 设备性能监测
- 研究数据质量保证

## English:

### Project Goals
MRI AutoQA aims to provide an automated, standardized MRI image quality analysis solution to help medical institutions and researchers evaluate MRI image quality metrics.

### Key Features
- **Automated Quality Analysis**: One-click batch MRI image quality assessment
- **Multi-dimensional Quality Metrics**: SNR, CNR, noise uniformity, artifact detection, etc.
- **Anatomical Region Adaptation**: Supports spine, brain, joints, and various anatomical regions
- **Sequence Type Recognition**: Automatically identifies T1, T2, PD, FLAIR sequences
- **Visualization Reports**: Generates detailed quality analysis reports with visualizations
- **Standardized Output**: Unified data format and report structure

### Use Cases
- Medical institution imaging quality control
- Clinical trial imaging quality assessment
- Equipment performance monitoring
- Research data quality assurance
